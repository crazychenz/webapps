Homework 3

Key Features

  • Class that takes integers as arguments and returns the integer products of the two integers.
  • The product method will return an integer of int type.
  • The main method will call the product method with the arguments from the command line and then print the result to STDOUT If the number is negative, it should be annotated with parenthesises instead of a minus sign.

Known Issues

  • There is no input validation on the arguments used from the command line.
